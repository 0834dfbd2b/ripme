ripme
=====

Client-side album ripper for various websites.

Written in Java for cross-platform support.

releases
========

**Early alpha release of [ripme.jar](http://rarchives.com/ripme.jar)**

Supported sites:
* imgur
* twitter
* tumblr
* instagram
* motherless
* imagefap
* imagearn
* seenive
* vinebox
* reddit
* gonewild
* 8muses
